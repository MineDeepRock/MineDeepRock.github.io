## 銃の仕様

#### 制圧射撃
半径3m以内に着弾すると制圧射撃の効果を受けます。  
3秒間、エフェクトが消え精度がガクンと落ちます。

#### オーバーヒート
特定の武器は打ち続けるとオーバーヒートを起こします。  
時間が立つと冷却され、再度撃つことができます。
